Цей проєкт створений для демонстрації роботи з Gulp: компіляція SCSS → CSS, автоматичне оновлення сторінки, додавання префіксів та мінімізація стилів.
Використані технології
- Gulp
- SCSS
- Autoprefixer
- BrowserSync
- CleanCSS
- Sourcemaps

Як запустити проєкт
1. Встановіть Node.js
2. Клонуйте репозиторій: git clone <link>
3. Встановіть залежності: npm install
4. Запустіть проект: gulp
5. Проєкт відкриється у браузері і буде автоматично оновлюватися.

Структура
src/scss/styles.scss — вихідні стилі
dist/css/styles.min.css — фінальний мініфікований CSS
index.html — сторінка проєкту
gulpfile.js — конфігурація Gulp

Ви можете переглянути фінальний проєкт за посиланням у розділі Pages.
